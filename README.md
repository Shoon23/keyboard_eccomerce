# keyboard_eccomerce

## Getting Started
    git clone <repo link>
    cd keyboard_eccomrce
    
    

### Frontend
    cd front-end
    npm install
    npm run dev
    frontend server will run at http://locahost:5173
    
### Backend
    open new terminal
    cd back-end
    npm install
    npm run dev  
    backend server will run at http://locahost:8080
